## The role

Ford VESSE (Vehicle Electrical, Software, Systems Engineering) is transitioning to a Software-Defined Vehicle (SDV) model. This TPM role leads the strategy, rapid prototyping, and adoption of AI tools that accelerate that transition — acting as translator, prototyper, and evangelist between VESSE engineers and the AI ecosystem.

## How this project is built

Claude Code and Gemini run simultaneously against the same `AGENTS.md` context. All design follows the Ford "Reimagine" design system (colors, typography, components) documented in `AGENTS.md`.
